generateShopperToken

  • generateShopperToken(contactId: string, options?: GenerateShopperTokenOptions): string
    • Defined in src/index.ts:141

    Generate a shopper token that can be used to make Lightrail calls restricted to that particular shopper. The shopper can be defined by the contactId, or an empty string for anonymous.

    eg: generateShopperToken("user-12345"); eg: generateShopperToken("user-12345", {validityInSeconds: 43200});

    Parameters

    • contactId: string

      the ID of the contact

    • Optional options: GenerateShopperTokenOptions

      the number of seconds the shopper token will be valid for OR a GenerateShopperTokenOptions object

    Returns string

    the shopper token
